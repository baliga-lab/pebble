Pebble - Welcome

About

Pebble is a repository for accessing experiment measurement data through an uniform interface. It consists of two parts:

  • A REST API to access the measurements in JSON format
  • A HTML based interface to allow viewing the data in the web browser

Pebble currently does not provide search services, it is expected to be used in combination with a search service like Echidna.

API

Retrieve JSON data

/pebble/api/1/<parameter-specification>

Data Matrix Viewer

/pebble/dmv/<parameter-specification>

Embedded Data Tables

/pebble/datatable/<parameter-specification>

HighCharts Data

/pebble/highcharts/<parameter-specification>

Parameter Specification

.../query?spec=<output-spec> (HTTP GET)
.../query (HTTP POST + Output specification as JSON)

JSON Query Format

General Structure

  [
    {
      "datasource": "sbeams"|"tiling"|"pre-sbeams"|"gwap"
      "params" : [type parameters]
      "conditions": [ <condition1>, ...]
    }
  ]

SBEAMS parameters:

  {
    "project_id": <id>
    "timestamp": <timestamp>
  }

Tiling parameters:

  {
    "timestamp": <timestamp>
  }
